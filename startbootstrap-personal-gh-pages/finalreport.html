Final Report

Overview of the Practicum Engagement

Biñan City Hall serves as the central administrative office of the City Government of Biñan, located in the province of Laguna, Philippines. It provides a wide range of services including infrastructure development, health and social welfare, disaster preparedness, and public information dissemination.

For this practicum engagement, I was part of a group of three members tasked with continuing the development of the official website of Biñan. The existing website required improvements in design, navigation, and content management to enhance effective public communication. Our group was assigned to further develop and refine the website, aiming to improve user experience, enhance accessibility to city services and information, and ensure efficient content updates.

The website is being developed using CodeIgniter 4 as the backend framework and MySQL as the database management system. My primary responsibilities focused on backend validation and user interface (UI) implementation, ensuring that user inputs were correctly processed and that the website provided a clean, responsive, and user-friendly experience. These tasks were critical to maintaining data accuracy and overall usability of the website.

My practicum rendered a total of 384 hours, starting from April 22, 2025, and is expected to be completed by July 28, 2025. This hands-on experience allowed me to apply academic knowledge in a real-world setting, enhance my programming and problem-solving skills, and contribute to an ongoing website project that will improve the online presence and operations of Biñan City Hall.

Detailed Discussion of Tasks and Assignments

As the backend developer and administrator for this project, my primary responsibilities focused on developing and managing the backend systems using CodeIgniter 4 PHP framework, with XAMPP serving as our local development environment. My role encompassed both backend logic implementation and administrative interface development, particularly focusing on the homepage content management and city officials module.

Backend Development:

  • Database Architecture: Designed and implemented MySQL database schemas using XAMPP's phpMyAdmin
  • API Development: Created RESTful APIs using CodeIgniter 4's API Response Trait for data handling
  • Data Validation: Implemented comprehensive server-side validation using CodeIgniter's Form Validation
  • Security Implementation: Set up authentication middleware and CSRF protection
  • Query Optimization: Developed efficient database queries using CodeIgniter's Query Builder

Admin Module Development:

  • Dashboard Interface: Created an intuitive admin dashboard for content management
  • City Officials Management: Developed CRUD functionality for managing official profiles and information
  • Content Management System: Built interfaces for updating homepage content dynamically
  • User Access Control: Implemented role-based access control for administrative functions
  • Data Management: Created tools for efficient data entry and updates through phpMyAdmin

Homepage Content Management:

  • Dynamic Content Integration: Implemented systems for updating homepage sections
  • Data Retrieval: Developed efficient methods for fetching and displaying content
  • Content Organization: Created structured content management for different homepage sections
  • Performance Optimization: Implemented caching and optimization techniques for faster page loads

Throughout the development process, I maintained a strong focus on code quality, system security, and performance optimization. The implementation utilized CodeIgniter 4's robust features for building secure and scalable web applications, while XAMPP provided reliable database management and testing capabilities.

Synthesis of the Practicum Engagement

During my practicum engagement on the Website of Biñan project, I acquired a range of valuable skills that significantly contributed to my personal and professional growth. A key area of development was strengthening my technical expertise, particularly in backend development and user interface design using CodeIgniter 4 and MySQL. I was responsible for the backend coding and the development of the admin page, which allowed me to deepen my understanding of data validation, system functionality, and responsive design principles. Working on a real-world website enhanced my ability to create efficient and secure backend systems.

Beyond technical skills, I honed my ability to collaborate effectively, working closely with two team members throughout the project. We consistently coordinated tasks, troubleshooted challenges together, and supported each other to achieve our objectives. This experience enhanced my communication skills and taught me the value of shared responsibility in a team setting.

Additionally, I improved my time management skills by managing deadlines, structuring development phases, and adapting to the professional environment of Biñan City Hall. This fostered greater discipline and focus in meeting project timelines and delivering quality outcomes.

Overall, these experiences have provided me with a solid foundation for my future career in the technology sector.
